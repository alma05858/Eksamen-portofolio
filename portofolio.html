PROJECT 03 - FUNDAMENTAL UX/UI

UX/UI

In project 3, we worked with UX/UI and learned about user experience, prototyping, usability, and testing. We applied a range of methods and tools in the design process, including desk research, inspiration research, idea development, mind mapping, brainstorming, sketching, mood boards, wireframes, and style tiles.

We also observed users to collect data for our own project websites. As part of the assignment, we were required to create a topic-based website, where we could freely choose the subject. In this process, we worked in Figma with wireframes and style tiles, and we used image editing tools such as Squoosh to edit and optimize images.

I developed a website using HTML, CSS, and JavaScript based on a digital prototype, which I tested using methods such as think-aloud testing and the 5-second test. Throughout the process, I documented the design and development stages, as part of the assignment was to demonstrate how ideas were transformed into concrete solutions. For this purpose, I created a process documentation as well as a presentation that outlines the key steps of the project.

PROJECT 04 - FUNDAMENTAL ANIMATION

GAME DESIGN

In project 4, we worked with basic animation and UI design, where we were introduced to design and composition principles in relation to creating game and UI elements. We learned to apply sketching and idea development techniques, and we worked with frontend technologies such as Adobe Illustrator, CSS, and JavaScript. Among other things, we learned about functions, variables, and events in JavaScript, as well as CSS positioning and animations, including how to use @keyframes to create movement and bring elements to life. Through creative exercises such as paper prototypes and hand-drawn characters, I developed my own game, Fairy Rescue, where I created all elements from scratch.

PROJECT 05 - FUNDAMENTAL CONTENT

WEBSITE REDESIGN

In project 5, Fundamental Content, we were introduced to key tools such as Git, GitHub, and Scrum, which we used in combination with Trello to plan and manage our project workflow. The main assignment was to propose a redesign of an existing company website. This required thorough research of the current solution, with a focus on structure and user experience. Based on the analysis, we developed wireframes and a style tile that communicated the desired visual identity, as well as a digital prototype in Figma.

We worked in teams and used a shared GitHub repository to collaborate on the code, where we continuously pushed and pulled updates so that the entire group had access to the latest version. Throughout the project, we gained practical experience in turning theory into practice — from idea development and collaboration to technical implementation and presenting our solution.

PROJECT 09 - 2ND SEMESTER EXAM

2ND SEMESTER EXAM

In the second semester, we worked on an exam project where we designed and developed a website for the café Arabica, with a focus on identity, community, and user experience.

We followed the Double Diamond model and worked with research, target group analysis, personas, wireframes, and an interactive prototype in Figma. The final solution was developed using Astro, HTML, CSS, and JavaScript, and we implemented dynamic features with Supabase. We also used Git and GitHub for collaboration and developed a social media strategy tailored to the target audience.

During this project, I gained a strong understanding of user-centered design and the interaction between UX/UI, development, and digital strategy.
